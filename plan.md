# 俄罗斯方块游戏开发计划

## 1. 项目概述
- **项目名称**: Tetris M25
- **项目类型**: Python桌面游戏
- **核心功能**: 经典俄罗斯方块游戏，支持简体中文界面
- **目标用户**: 休闲游戏玩家

## 2. 技术选型
- **语言**: Python 3.8+
- **GUI框架**: Pygame (成熟稳定，跨平台支持好)
- **打包工具**: PyInstaller (支持多平台构建)

## 3. 功能规划

### 3.1 核心游戏功能
- [ ] 7种经典方块类型 (I, O, T, S, Z, J, L)
- [ ] 方块自动下落
- [ ] 方块左右移动
- [ ] 方块旋转
- [ ] 方块快速下落
- [ ] 方块硬下落 (直接落底)
- [ ] 消行功能 (1-4行同时消除)
- [ ] 分数计算系统
- [ ] 等级系统 (每10行升一级)
- [ ] 游戏结束检测

### 3.2 界面功能
- [ ] 主菜单界面 (开始游戏、设置、退出)
- [ ] 游戏界面 (游戏区域、分数、等级、下一个方块预览)
- [ ] 游戏结束界面 (显示最终分数)
- [ ] 简体中文界面

### 3.3 控制系统
- [ ] 方向键控制移动和旋转
- [ ] 空格键硬下落
- [ ] P键暂停/继续
- [ ] ESC键返回菜单

### 3.4 辅助功能
- [ ] 日志系统 (保存到游戏目录的tetris.log)
- [ ] 音效系统 (可选)
- [ ] 设置保存 (难度、按键等)

## 4. 界面设计

### 4.1 布局
```
┌─────────────────┬──────────┐
│                 │  NEXT    │
│   游戏区域      │  方块    │
│   10x20        │  预览    │
│                 ├──────────┤
│                 │  SCORE   │
│                 │  LEVEL   │
│                 │  LINES   │
└─────────────────┴──────────┘
```

### 4.2 配色方案
- 深色背景 (护眼)
- 彩色方块 (每种类型不同颜色)
- 霓虹风格边框

## 5. 构建与发布
- [ ] GitHub Actions配置 (Windows, Linux, macOS)
- [ ] 多平台构建产物
- [ ] checksum.txt生成
